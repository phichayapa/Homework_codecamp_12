ข้อ1 ค้นหาข้อมูลว่าแท็ก <!doctype> มีหน้าที่อะไร?

คือ การประกาศชนิดของเอกสาร ในบรรทัดแรกสุดของเอกสาร Html จะเป็นการประกาศชนิดของเอกสาร หรือ Document Type Declaretion (DTD) หรือเรียกสั้น ๆ ว่าการประกาศ DOCTYPE นั้นเอง ซึ่งมาตราฐานที่เราใช้กันทุกวันนี้กับการแสดงผลเว็บเพจ จะมีที่ใช้กันอยู่ 3 มาตราฐาน คือ - HTML - XHTML - HTML 5

ข้อที่3 แท็ก <b >, <i > ต่างจาก <strong >,<em > หรือไม่ อย่างไร?

เหมือนกัน แต่แท็ก <strong>, <em> จะให้ความสำคัญกว่า แท็ก <b >, <i >

แท็ก <b> - Bold text เป็นตัวหนาทั่วๆไป

แท็ก <i > - Italic text ข้อความเป็นตัวเอียง

แท็ก <strong> - Important text จะเน้นในหัวข้อที่สำคัญ อาจเป็น Keyword ให้ความสำคัญกว่าแท็ก <b>

แท็ก <em> - Emphasized text ข้อความเป็นตัวเอียง แต่ควรใช้กว่าแท็ก <i >
